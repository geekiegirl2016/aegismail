# Code of Conduct

The AegisMail project adopts the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) as its Code of Conduct.

## Our standards

We expect everyone participating in AegisMail — contributors, maintainers, and users interacting in our spaces — to help create a welcoming, respectful, and harassment-free community. Treat others with kindness, assume good faith, give and accept constructive feedback gracefully, and prioritize what is best for the project and its users.

Behavior that violates the Contributor Covenant will not be tolerated in any AegisMail space (issues, pull requests, discussions, chat, or events).

## Scope

This Code of Conduct applies within all project spaces and also applies when an individual is officially representing the project in public spaces.

## Reporting

If you experience or witness conduct that you believe violates this Code, please report it by emailing **conduct@aegismail.org**. Reports will be handled confidentially by the project maintainers.

See the full Contributor Covenant text and the enforcement guidelines at:
<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>
